# Cursor

Shared Cursor (VS Code) configuration and utilities.

Contents
- `settings.json`: Base Cursor settings (JSONC; comments allowed).
- `keybindings.json`: Shared keybindings.
- `local-themes/`: Optional local theme VSIX (no marketplace install required).
- `themes-no-extensions/`: Theme override JSON plus an apply script that merges into `settings.json`.
